# Local Theory v2

## Mission
Rebuild the digital agency dashboard with a premium, high-performance stack. Replace Base44 with Next.js + Supabase.

## Stack
- **Framework:** Next.js 14
- **UI:** Tailwind CSS, Framer Motion, Lucide Icons (Premium Glass/Dark Mode)
- **Backend:** Supabase (PostgreSQL, Auth)
- **Hosting:** Vercel

## Core Features (Migration List)
1.  **Agency Dashboard:** Admin view of all clients and stats.
2.  **Client Portal:** Secure login for clients to see their projects.
3.  **AI Audits:** Generate and view SEO/Performance reports.
4.  **Project Boards:** Kanban style task management.
5.  **Proposals:** Create and send contracts/proposals.
6.  **Content Calendar:** Schedule and plan social posts.
